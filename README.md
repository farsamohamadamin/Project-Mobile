# 📝 Todo List App

A simple and lightweight **Todo List** application built with **.NET MAUI** and **C#**.

This project allows users to manage their daily tasks by adding and deleting items. Tasks are automatically stored on the local device, so they remain available even after the application is closed.

---

## ✨ Features

- ➕ Add new tasks
- 🗑️ Delete existing tasks
- 💾 Automatic local data storage
- 📂 Persistent task saving
- 🌙 Simple dark-themed user interface
- 📱 Cross-platform application with .NET MAUI

---

- .NET MAUI

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/farsamohamadamin/Project-Mobile.git
```

### Open the project

1. Install **Visual Studio 2022**
2. Install the **.NET MAUI** workload
3. Open the solution
4. Build and run the application

Supported platforms:

- Android
- Windows
- iOS
- macOS

---

## 💾 Data Storage

The application stores all tasks locally using the application's data directory, ensuring that your tasks remain available after restarting the app.
