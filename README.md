# 📝 Todo List App

A simple and modern **Todo List** application built with **.NET MAUI**
and **C#**.

This application allows users to add and delete daily tasks while
storing them locally so they remain available after restarting the app.

## ✨ Features

-   Add new tasks
-   Delete tasks
-   Automatic local storage
-   Persistent task saving
-   Cross-platform with .NET MAUI
-   Clean dark UI

## 🛠️ Technologies

-   .NET MAUI
-   C#
-   XAML
-   Local File Storage

## 🚀 Getting Started

``` bash
git clone https://github.com/farsamohamadamin/Todo-List.git
```

Open the project in **Visual Studio 2022** with the **.NET MAUI**
workload installed.
